Usage Guidelines

Do's ✅

  • Use United Red for primary CTAs and key elements
  • Use Gold sparingly for awards, highlights, special features
  • Maintain high contrast (white on black, white on red)
  • Use font-bold (700) or font-black (900) for headings
  • Give buttons generous padding and hover states
  • Use consistent spacing (multiples of 4px)

Don'ts ❌

  • Don't use red on black for body text (low contrast)
  • Don't overuse gold (loses impact)
  • Don't use font weights below 400 (too thin)
  • Don't use gray text on colored backgrounds
  • Don't make clickable elements smaller than 44×44px (mobile)

Accessibility

  • All text meets WCAG AA contrast requirements
  • Buttons have minimum 44px touch targets
  • Focus states visible for keyboard navigation
  • Color is never the only indicator of meaning
